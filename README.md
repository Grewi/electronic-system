# Electronic-system

## Установка системы
В директории проекта создать папку system. Например:
```php
mkdir system
```

Перейти в созданную директорию
```php
cd system
```
Запустить установщик
```php
php install
```
Установщик создаст необходимую структуру файлов и первоначальные данные конфигурации.
